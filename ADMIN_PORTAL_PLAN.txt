# PrintRUSH Lopez — Admin Portal, Sandbox Integrations & Plan Strategy

This document captures the current refactor direction for the admin portal and the new sandbox replacements for PayMongo and Shipmates.

## 1. Admin Portal Strategy

A Super Admin Portal is the right approach to manage multiple shops without touching the Supabase SQL Editor.

### Architecture

- **Authentication & Roles**
  - Add a `platform_admins` table linked to `auth.users`.
  - Admins log in at `/admin/login.html`.
  - Access to `/admin/dashboard.html` is limited to users in `platform_admins`.
- **Shop & Subscription Management**
  - Shop list with `active/inactive` toggle, plan assignment, and subscription renewal dates.
  - Add/edit shop details such as name, address, delivery fees, and plan tier.
  - Add `subscription_expires_at` to the `shops` schema for renewal tracking.
- **System Health & Logs**
  - Display recent payment simulation events and delivery booking status.
  - Add a live error feed for sandbox flow failures.
- **Admin Management**
  - Invite and manage Super Admins from the portal.
  - Role support for `admin` and `super_admin`.

## 2. Sandbox Payment + Delivery Simulation

We are replacing real Shipmates/PayMongo dependencies with a fully internal sandbox that:

- mimics the same API flow structures, but never calls external APIs
- stores payments, orders, and delivery records in Supabase
- simulates payment states (`pending → paid / failed`)
- simulates delivery stages (`pending → picked_up → in_transit → delivered`)
- uses timestamp-driven progression for realistic timelines

### Implementation Highlights

- `create-payment` Edge Function now creates a sandbox payment record and returns a local checkout URL.
- `mock-payment-status` Edge Function advances sandbox payments based on `expected_paid_at`.
- `dispatch-delivery` Edge Function now creates a mock delivery booking with tracking, timeline milestones, and a simulated courier.
- `delivery-status` Edge Function keeps delivery stage progression in sync when customers open tracking.
